# Rozana_Grocery (e commers services application)
# **Description:** Rozana Grocery Ecommerce App is a Flutter-based grocery shopping app with a clean UI, smooth state management, and REST API integration. It offers cart storage, custom themes, and localization. Users can browse groceries with details and reviews, add items to the cart, purchase products securely using UPI or cash-on-delivery, and easily manage their orders and payments.

# Technology Used :  Dart, Flutter, Third Party Rest API, GetX 

# IDE Used : Android Studio 

# Duration : Devlopments phase
