# Rozana_Grocery (E-Commerce Services Application)

## Description
Rozana Grocery is a Flutter-based e-commerce application designed for grocery shopping with a clean, modern UI and smooth state management.  
It integrates REST APIs to provide real-time product data and supports features like cart storage, custom themes, and localization.  

Users can:
- Browse grocery products with detailed descriptions and reviews  
- Add or remove items from the cart  
- Purchase products securely via UPI or Cash on Delivery (COD)  
- Manage their orders and payments with ease
